Sobre mim

Olá, meu nome é Camila e atualmente estou dedicada aos estudos na área de programação, com especial interesse na linguagem Java. Tenho uma forte afinidade com disciplinas exatas, como matemática e física, que sempre despertaram em mim o desejo de compreender estruturas profundas e resolver problemas de forma lógica e criativa. Além do universo tecnológico, cultivo uma paixão pela literatura — especialmente a russa e a japonesa — sendo Dostoiévski uma das minhas maiores referências. A complexidade psicológica e filosófica presente em suas obras influencia diretamente minha forma de pensar e abordar desafios. Também me dedico à pintura com tinta a óleo, uma atividade que me proporciona equilíbrio, concentração e sensibilidade estética. Acredito que a programação é, acima de tudo, uma forma de expressão intelectual. É nesse espaço entre a racionalidade das ciências exatas e a subjetividade das artes que busco construir minha trajetória como desenvolvedora, aliando técnica, criatividade e profundidade.

Portfólio

Este é um projeto de um site chamado mario.site, criado como um estudo para ser uma enciclopédia sobre o universo do Super Mario. O site é interativo e informativo, com seções que mostram jogos e personagens em carrosséis, além de uma área para inscrição em uma newsletter. Ele foi desenvolvido com tecnologias front-end como HTML, CSS e JavaScript, incluindo a biblioteca Swiper.js para os carrosséis.

Esta é uma aplicação de backend em Java e Spring Boot para gerenciar inscrições em eventos. O sistema permite que um usuário se inscreva em um evento e, caso ainda não tenha cadastro, um novo perfil é criado automaticamente. Uma funcionalidade central é o sistema de indicações, que registra quem indicou um novo participante. Com base nessas indicações, a aplicação gera um ranking de participantes para cada evento, permitindo também a consulta da posição específica de um usuário nesse ranking. Todas as informações de eventos, usuários e inscrições são armazenadas em um banco de dados MySQL.

Esta é uma aplicação de backend, desenvolvida em Node.js e TypeScript, que funciona como uma API para o gerenciamento de cursos. O sistema é construído sobre o framework Fastify, escolhido por sua alta performance no tratamento de requisições HTTP, e utiliza o Drizzle ORM para interagir com um banco de dados PostgreSQL. Além disso, a aplicação garante a integridade dos dados que chegam através da API por meio de validações de esquema com a biblioteca Zod. O projeto também adota boas práticas de desenvolvimento, contando com uma suíte de testes automatizados com Vitest, o que assegura a qualidade e a confiabilidade do código.
